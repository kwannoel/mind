---
title: Common asymptotic growth functions
date: "2020-08-10"
---

| Function         | Name        |
|------------------|-------------|
| 1                | constant    |
| $lg(lg(n))$      | log log     |
| $lg(n)$          | log         |
| $n^c, 0 < c < 1$ | sublinear   |
| n                | linear      |
| $n lg(n)$        | n log n     |
| $n^2$            | quadratic   |
| $n^3$            | cubic       |
| $n^k, k \geq 1$  | polynomial  |
| $c^n$            | exponential |
| n!               | factorial   |
