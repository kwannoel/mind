# Why use Python?

<96995f76>
- Beautiful is better than ugly.
- Explicit is better than implicit.
- Simple is better than complex.
- Complex is better than complicated.
- Readability counts.

Python Standard library has many convenient builtins:
- HTTP
- MIME
- GUI
- rng
- relational db
- regex
- unit tests
- floats
