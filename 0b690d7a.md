# Mips Memory element

| Command              | Q(t) | Q(t + 1) |
|----------------------|------|----------|
| Set                  | X    | 1        |
| Reset                | X    | 0        |
| Memorise / No Change | 0    | 0        |
| Memorise / No Change | 1    | 1        |

Note that there is a `clock` input (t).

`clock` is a square wave.

```
1   ------    ------
    |    |    |
0 ---    ------
```
