# Packet transmission delay

The lifetime of a packet is as follows:

```
packet -> link -> ...
```

We measure the time taken to push the entire packet onto the link, i.e.

```
packet -> link
```

**time needed to transmit L-bit packet into link = L(bits) / R (bits/sec)**

A.k.a. [[ec8c89c0]]
